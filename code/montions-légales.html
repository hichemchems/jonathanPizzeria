Politique de confidentialité

Les données personnelles collectées sur le site internet Chez Jonathan sont traitées par la société Jonathan Pizza.

Les données collectées sont utilisées pour :

  • Gérer les commandes et les livraisons
  • Envoyer des informations commerciales
  • Améliorer l'expérience des utilisateurs

Les données personnelles sont conservées pendant une durée de 3 ans à compter de la dernière interaction avec le site internet.

Vous disposez des droits suivants sur vos données personnelles :

  • Droit d'accès
  • Droit de rectification
  • Droit à l'effacement
  • Droit à la limitation du traitement
  • Droit à la portabilité des données
  • Droit d'opposition

Pour exercer vos droits, vous pouvez contacter la société Jonathan Pizza par email à l'adresse contact@chez-jonathan.fr.

Responsabilités

La société Jonathan Pizza s'engage à fournir des informations fiables et à jour sur son site internet. Toutefois, il est possible que des erreurs ou des omissions se produisent. En cas de constatation d'une erreur ou d'une omission, veuillez nous en informer afin que nous puissions la corriger.

La société Jonathan Pizza ne saurait être tenue responsable des dommages directs ou indirects qui pourraient résulter de l'utilisation de son site internet.

La société Jonathan Pizza ne saurait être tenue responsable du contenu des sites internet tiers auxquels son site internet renvoie.

Droits des utilisateurs

Les utilisateurs ont le droit d'accéder aux données personnelles qui les concernent. Ils peuvent en demander une copie à tout moment.

Les utilisateurs ont le droit de rectifier les données personnelles qui les concernent si elles sont inexactes ou incomplètes.

Les utilisateurs ont le droit d'effacer leurs données personnelles, dans certains cas, tels que si les données ne sont plus nécessaires pour les finalités pour lesquelles elles ont été collectées ou si l'utilisateur a retiré son consentement.

Les utilisateurs ont le droit de demander la limitation du traitement de leurs données personnelles, dans certains cas, tels que si l'utilisateur conteste l'exactitude des données ou s'oppose à leur traitement.

Les utilisateurs ont le droit de recevoir les données personnelles qu'ils ont fournies à un responsable du traitement dans un format structuré, couramment utilisé et lisible par machine. Ils ont également le droit de transmettre ces données à un autre responsable du traitement.

Les utilisateurs ont le droit de s'opposer au traitement de leurs données personnelles, dans certains cas, tels que si le traitement est basé sur l'intérêt légitime du responsable du traitement.

Sécurité des données

La société Jonathan Pizza s'engage à protéger les données personnelles des utilisateurs. Elle prend toutes les mesures de sécurité nécessaires pour empêcher l'accès non autorisé aux données ou leur divulgation.

La société Jonathan Pizza utilise des méthodes de cryptage pour protéger les données personnelles lors de leur transmission sur son site internet.

La société Jonathan Pizza dispose d'une politique de sécurité des données qui définit les mesures de sécurité mises en place pour protéger les données personnelles.

Transferts internationaux de données

La société Jonathan Pizza peut transférer des données personnelles vers des pays situés en dehors de l'Union européenne. Dans ce cas, elle s'engage à prendre toutes les mesures nécessaires pour garantir un niveau de protection adéquat des données personnelles.

La société Jonathan Pizza peut transférer des données personnelles vers des pays situés en dehors de l'Union européenne uniquement si le pays de destination offre un niveau de protection adéquat des données personnelles.
